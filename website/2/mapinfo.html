Map powered by Google
Instructions:

- Click on white markers to see tags
- Use location button so see tag on GoogleMaps
- Use tweet button so tweet this tag
- DoubleClick to Zoom-In
- CTRL + DoubleClick to Zoom-Out
- Mouse - RightClick to Zoom-Out
- Mouse - Use Trackwheel to Zoom
- MacBook - Drag with two fingers on trackpad up or down to Zoom
- Drag to Pan
- Click on points to see drawings
- Zoom right in to find more drawings at same location
- If at same location, map shows the latest drawings top


Thanks a lot to Andrey Yazev for the GoogleMap integration.
