SKTCH Desktop is an application for both Mac and Windows which allows you to view, edit and export the drawings you made using the iPhone application.

Features
- Open .xml files sent from iPhone SKTCH app
- Edit points by pressing space on the keyboard
- Save settings
- Save high resolution images
- Import images as backgrounds
- SKTCH Remote - iPhone application to draw on the desktop app. Download from AppStore

Keyboard Shortcuts
'1' - settings panel
'2' - preset panel
'3' - random
's' - save settings
'c' - clear screen
'x' - save image
'space' - edit points

(Mac OSX 10.5 and above // Windows version coming soon)
